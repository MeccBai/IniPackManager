# Bun Command Policy

Use Bun for all JavaScript and TypeScript commands in this repository.

- Install dependencies with `bun install`.
- Run package scripts with `bun run <script>`; for example, `bun run build`.
- Run one-off package executables with `bunx <package>`.
- Do not use `npm`, `npx`, `node`, `yarn`, or `pnpm` commands.
- Keep the existing Cargo and Tauri commands unchanged; this policy applies only to JavaScript and TypeScript tooling.
