# Tauri Updater

This project is configured to use the Tauri updater plugin with a public JSON manifest.

Configured endpoint:

`https://github.com/MeccBai/IniPackManager/releases/latest/download/latest.json`

What you need to replace:

- `src-tauri/tauri.conf.json` -> `plugins.updater.pubkey`
- `docs/updater/latest.json.example` -> all placeholder values

Recommended GitHub Release layout:

- Upload the updater manifest as `latest.json`
- Upload the signed Windows updater artifact generated by Tauri
- Upload the matching `.sig` file and copy its contents into `latest.json`

Important:

- Tauri updater does not use an SSH public key like `ssh-ed25519 ...`
- You must generate a Tauri updater signing keypair, for example with `pnpm tauri signer generate` or `npx tauri signer generate`
- Put the generated updater public key into `plugins.updater.pubkey`

If GitHub Release assets are not suitable, point `endpoints[0]` to any public HTTPS JSON URL, for example a Cloudflare Worker endpoint serving the same manifest schema.
